## silo tools

快速构建 webpack + karma 工具

### 安装

```shell
 npm install silo-tools -g
```

### 简单开始

```shell

silo init myProject  初始化项目

npm install

npm start 运行本地服务器

```

### 看看你代码到底写了多少行了（码农需要一点成就感）

- 运行 `npm run len:all`

### 生成es6文档

- 运行 `npm run doc`

### 如何运行测试

- 运行 `npm run test`

### 如何运行测试覆盖率

- 运行 `npm run test:cov`

### 运行karma多浏览器测试

- 运行 `npm run karma`

### 更多命令

通过 `silo --help` 查看
